# 📝 CodeAsire React Notes App

This project is part of my **Front-End Web Developer Internship at CodeAsire**.

---

## 🚀 Project Overview

A dynamic Notes Application built using **React.js**, allowing users to create, delete, and manage notes efficiently with real-time updates.

---

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- LocalStorage

---

## ✨ Features

- Add notes dynamically
- Delete notes instantly
- Real-time UI updates using React state
- Persistent data using localStorage
- Component-based architecture
